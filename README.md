# EventReg - React
## Description

Initial scope is to register an attendee for an event.

Future features may include:
* Multi-page registration
  * Demos
  * Items
  * Sessions
  * Visa
  * Invite a friend
  * Payment
* Registrant admin: Update record
* Upgrade / Add items
* Exhibitor booth staff registration
* Event admin
  * Create, View, Edit, Delete
